== Copy-Paste Templates

=== Template 1: Mixed Content (Recommended)

[source,asciidoc]
----
++++
<div class="browser-window" style="min-height: 450px;">
  <div class="browser-window-header">
    <div class="browser-window-buttons">
      <span class="browser-window-dot"></span>
      <span class="browser-window-dot"></span>
      <span class="browser-window-dot"></span>
    </div>
    <div class="browser-window-address-bar">YOUR_URL_HERE</div>
    <div class="browser-window-menu-icon">
      <span class="browser-window-menu-bar"></span>
      <span class="browser-window-menu-bar"></span>
      <span class="browser-window-menu-bar"></span>
    </div>
  </div>
  <div class="browser-window-body">
++++

image::YOUR_IMAGE.png[Your Description, role="browser-screenshot"]

++++
  </div>
</div>
++++
----

=== Template 2: Pure HTML

[source,asciidoc]
----
++++
<div class="browser-window" style="min-height: 450px;">
  <div class="browser-window-header">
    <div class="browser-window-buttons">
      <span class="browser-window-dot"></span>
      <span class="browser-window-dot"></span>
      <span class="browser-window-dot"></span>
    </div>
    <div class="browser-window-address-bar">YOUR_URL_HERE</div>
    <div class="browser-window-menu-icon">
      <span class="browser-window-menu-bar"></span>
      <span class="browser-window-menu-bar"></span>
      <span class="browser-window-menu-bar"></span>
    </div>
  </div>
  <div class="browser-window-body">
    <img src="_images/YOUR_IMAGE.png" alt="Your Description" class="browser-screenshot" />
  </div>
</div>
++++
----

Replace:
- `YOUR_URL_HERE` with your desired URL
- `YOUR_IMAGE.png` with your image filename
- `Your Description` with appropriate alt text
